# EcoPoint

EcoPoint é um aplicativo desenvolvido em React Native com Expo, voltado para auxiliar usuários na localização e gerenciamento de pontos de descarte consciente.

## Tecnologias utilizadas

- React Native
- Expo
- Expo Router
- TypeScript

## Como executar

npm install
npx expo start
